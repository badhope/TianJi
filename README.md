# 灵墟 (LingXu)

末法时代·失落修行文明档案馆

## 技术栈

- Astro 4.x
- TypeScript
- Tailwind CSS
- Astro Content Collections
- MDX

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 项目结构

```
src/
├── components/     # 组件
│   ├── layout/     # 布局组件
│   ├── ui/         # UI组件
│   ├── interactive/ # 交互组件
│   └── widgets/    # 小部件
├── layouts/        # 页面布局
├── pages/          # 页面
│   ├── archive/    # 档案馆
│   ├── medicine/   # 医药
│   ├── myth/       # 神话
│   ├── dharma/     # 法脉
│   └── realms/     # 境界
├── content/        # 内容配置
├── styles/         # 全局样式
├── lib/            # 工具函数
└── data/           # 静态数据
```
