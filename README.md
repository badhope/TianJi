# 灵墟

**末法时代 · 失落修行文明档案馆**

> 天地灵气衰竭，法脉传承断绝。唯有典籍文字，传其余绪。

## 项目简介

灵墟，取"灵炁墟烬"之意，是一个专注于中国修行文明的数字档案馆。我们致力于在典籍、宗教、民俗、传说与小说之间，勾勒出一个曾经真实存在过的修行世界。

这不是游戏攻略，也不是玄幻小说设定集。这是一个严肃的、关于真实历史与文化中被遗忘之物的档案项目。

## 核心内容

### 📜 档案馆 `/archive`

收集、整理、研究散落于各处的修行文明碎片。

### 🏺 医药 `/medicine`

从《黄帝内经》探索身体与能量的奥秘，从传统医学理解修行基础。

### 🗺️ 神话 `/myth`

将《山海经》重新诠释为失落时代的地理档案，追溯神怪异兽的真相。

### 🔮 法脉 `/dharma`

梳理佛教东渐与道教兴起的修行传承脉络，从佛道典籍中探寻法脉。

### ⚔️ 境界 `/realms`

研究修行境界的演化，理解从炼体到飞升的完整路径。

## 设计理念

### 视觉风格

- **神秘档案氛围**：深邃的暗色调搭配金色点缀，营造古籍珍藏的感觉
- **东方美学**：融合传统水墨意境与现代简约设计
- **克制冷艳**：避免廉价网游感，追求高端、克制、沉浸式的文化体验

### 字体选择

- **标题字体**：站酷高端黑、阿里巴巴普惠体等
- **正文字体**：思源宋体、Noto Serif SC，保留古典韵味同时确保可读性
- **装饰字体**：可用于特殊强调

### 动效设计

- 优雅克制，不喧宾夺主
- 粒子飘浮、神圣几何闪烁等氛围元素
- 滚动渐入、悬停反馈等交互反馈
- 支持减少动画偏好

## 技术架构

- **框架**：Astro 静态网站框架
- **语言**：TypeScript
- **样式**：Tailwind CSS
- **内容**：MDX 格式，支持富文本内容
- **部署**：GitHub Pages

## 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
